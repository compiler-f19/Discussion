這學期間遇到的所有問題都會被放在這個 Repo 的`Issues` 區域，如果想發問就直接開一個 `Issue` 出來吧!
討論時可以直接標記同學的帳號 (比如 `@cmprfk1` ) 讓對方收到通知。也可以用團隊標記 `@2019_taching_team` 讓所有助教跟老師收到通知。

大部分已解決問題會被自動歸類在 `closed issued` 之下，所以別忘記先查看已經關閉的 `Issues`。
作業快寫完, 早早出去玩!
